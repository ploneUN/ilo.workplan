ilo.workplan Installation
-------------------------

To install ilo.workplan using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ilo.workplan`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ilo.workplan

* Re-run buildout, e.g. with:

    $ ./bin/buildout
